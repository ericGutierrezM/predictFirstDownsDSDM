about the library.

firstDown is a Python library envisioned to get the most out of the vast amount of sports-related data available! Although it has been designed to predict first downs (play success) in the NFL, its structure allows for changes and additions that make possible to perform data analysis on (virtually) any sports. So no matter if it's an extra yard, mile, goal, or point, firstDown will get you there!

Check out firstDown on GitHub!
